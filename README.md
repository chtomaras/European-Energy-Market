# European Energy Market Analysis Web App

[cite_start]Αυτή η εφαρμογή αναπτύχθηκε στα πλαίσια του μαθήματος "Εισαγωγή στην Επιστήμη του Ηλεκτρολόγου Μηχανικού" στο Πανεπιστήμιο Πατρών. [cite_start]Επιτρέπει τη μελέτη, σύγκριση και ανάλυση των τιμών ηλεκτρικής ενέργειας στην ελληνική και ευρωπαϊκή αγορά.

## 🚀 Λειτουργίες

* [cite_start]**Ανάλυση Μιας Χώρας:** Αναζήτηση τιμών ενέργειας για επιλεγμένα χρονικά διαστήματα (έως ένα έτος) με υπολογισμό δεικτών όπως Μέσος Όρος και Τυπική Απόκλιση[cite: 26, 28].
* [cite_start]**Σύγκριση Περιόδων:** Σύγκριση τιμών της ελληνικής αγοράς μεταξύ διαφορετικών χρονικών περιόδων[cite: 22].
* [cite_start]**Σύγκριση Χωρών:** Ταυτόχρονη σύγκριση τιμών μεταξύ διαφορετικών ευρωπαϊκών χωρών για το ίδιο χρονικό διάστημα, αναδεικνύοντας τις επιπτώσεις παγκόσμιων γεγονότων στην αγορά[cite: 13].

## 🛠️ Τεχνολογίες

* [cite_start]**Backend:** Python με το framework **Flask**[cite: 1, 49].
* [cite_start]**Frontend:** HTML5, CSS3, Tailwind CSS και JavaScript[cite: 12, 18, 28].
* [cite_start]**Δεδομένα:** Ανάκτηση δεδομένων σε πραγματικό χρόνο από το **ENTSO-E API** (Restful API)[cite: 14, 15].
* [cite_start]**Γραφήματα:** Χρήση της βιβλιοθήκης **Chart.js** για τη δημιουργία δυναμικών διαγραμμάτων[cite: 12, 28].
