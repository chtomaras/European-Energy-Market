# European Energy Market Analysis Web App

Αυτή η εφαρμογή αναπτύχθηκε στα πλαίσια του μαθήματος "Εισαγωγή στην Επιστήμη του Ηλεκτρολόγου Μηχανικού" στο Πανεπιστήμιο Πατρών. Επιτρέπει τη μελέτη, σύγκριση και ανάλυση των τιμών ηλεκτρικής ενέργειας στην ελληνική και ευρωπαϊκή αγορά.

## 🚀 Λειτουργίες

* **Ανάλυση Μιας Χώρας:** Αναζήτηση τιμών ενέργειας για επιλεγμένα χρονικά διαστήματα (έως ένα έτος) με υπολογισμό δεικτών όπως Μέσος Όρος και Τυπική Απόκλιση.
* **Σύγκριση Περιόδων:** Σύγκριση τιμών της ελληνικής αγοράς μεταξύ διαφορετικών χρονικών περιόδων.
* **Σύγκριση Χωρών:** Ταυτόχρονη σύγκριση τιμών μεταξύ διαφορετικών ευρωπαϊκών χωρών για το ίδιο χρονικό διάστημα, αναδεικνύοντας τις επιπτώσεις παγκόσμιων γεγονότων στην αγορά.

## 🛠️ Τεχνολογίες

* **Backend:** Python με το framework **Flask**.
* **Frontend:** HTML5, CSS3, Tailwind CSS και JavaScript.
* **Δεδομένα:** Ανάκτηση δεδομένων σε πραγματικό χρόνο από το **ENTSO-E API** (Restful API).
* **Γραφήματα:** Χρήση της βιβλιοθήκης **Chart.js** για τη δημιουργία δυναμικών διαγραμμάτων.
